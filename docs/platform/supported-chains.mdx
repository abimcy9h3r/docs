# Supported Chains

ForLarge is a multi-chain marketplace. Creators and buyers can transact on multiple blockchain networks.

## Live Chains

### Solana

Solana is a high-performance Layer 1 blockchain known for fast transactions and low fees.

**Why Solana?**
- Sub-second finality
- Transaction fees under $0.01
- Massive ecosystem with 50M+ active wallets
- Native USDC support via SPL token standard

**Technical Details:**
- **Network**: Solana Mainnet Beta
- **USDC Contract**: `EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v`
- **RPC Provider**: Helius
- **Supported Wallets**: Phantom, Solflare (via WalletConnect)

### Injective

Injective is a Cosmos SDK-based blockchain optimized for finance and trading applications.

**Why Injective?**
- Fast finality (~2 seconds)
- Low transaction fees
- Growing DeFi ecosystem
- IBC-connected to 50+ Cosmos chains

**Technical Details:**
- **Network**: Injective Mainnet
- **USDC Denom**: `peggy0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48`
- **Supported Wallets**: Keplr, Leap

## Coming Soon

### Base

Base is a Layer 2 Ethereum network built by Coinbase.

**Why Base?**
- Low fees (L2 rollup on Ethereum)
- Direct Coinbase Wallet integration
- Large user base from Coinbase ecosystem
- Native USDC support

**Status**: Under development. ETA will be announced.

## Chain Selection

When purchasing a product, buyers choose which chain to pay on. Creators set their wallet addresses for each supported chain.

<Note>
  Not all creators accept payments on all chains. Check the product listing to see which chains are available.
</Note>

## Verification

ForLarge verifies every transaction on-chain before issuing download links:

| Chain | Verification Method | Typical Time |
|-------|-------------------|--------------|
| **Solana** | RPC `getTransaction` via Helius | 1-3 seconds |
| **Injective** | LCD REST API query | 2-5 seconds |

All verification happens server-side. The buyer's client never self-reports payment — ForLarge independently confirms every transaction.
