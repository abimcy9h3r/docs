# Downloading Products

After completing a purchase on ForLarge, you'll receive a secure download link.

## How Downloads Work

1. **Payment verified** — ForLarge confirms your transaction on-chain
2. **Token generated** — a unique, encrypted download token is created
3. **Redirect** — you're automatically sent to the download page
4. **Email receipt** — a confirmation email with the download link is also sent

## Download Security

ForLarge uses several layers of security to protect creators' content:

- **Unique tokens** — each purchase generates a cryptographically random download token
- **Time-limited** — download links expire after a set period
- **Single-use** — tokens are invalidated after use
- **Server-verified** — every download request is validated against the purchase record

<Warning>
  Download your product as soon as possible after purchase. Links expire for security.
</Warning>

## Accessing Your Download

### From the Checkout Flow

After a successful payment, you'll be automatically redirected to the download page. Click the **Download** button to save the file.

### From Email

Check the email address you provided at checkout. You'll receive:
- Purchase confirmation
- Product details
- A secure download link

<Info>
  Don't see the email? Check your spam/junk folder. Emails are sent from `forlarge.app`.
</Info>

## Troubleshooting Downloads

<AccordionGroup>
  <Accordion title="Download link expired">
    Download links are time-limited for security. Contact the creator with your transaction hash to request a new link.
  </Accordion>
  <Accordion title="File won't open">
    Make sure you have the appropriate software for the file type. ZIP files need to be extracted first. WAV/MP3 files open in any audio player.
  </Accordion>
  <Accordion title="Download is slow or interrupted">
    Try again with a stable internet connection. Product files are served from fast CDN infrastructure.
  </Accordion>
  <Accordion title="Wrong file received">
    Contact the creator directly. They can verify the product file and send you the correct version.
  </Accordion>
</AccordionGroup>

## File Types You May Receive

| Type | Extension | How to Open |
|------|-----------|-------------|
| Audio | .mp3, .wav, .flac | Any music player |
| Archive | .zip, .rar | Extract with built-in OS tools or 7-Zip |
| Document | .pdf | Any PDF reader |
| Image | .png, .jpg, .psd | Image viewer or Photoshop |
